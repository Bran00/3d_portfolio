### Instalation 

Use this project with:

[First pass] Clone the repository: `git clone https://github.com/`
[Second pass] Make the install the dependecies:

---------------------
npm install
---------------------
[third pass] Make the aplicattion run:

--------------------
npm run dev
---------------------
